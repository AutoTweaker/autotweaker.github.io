AgentAPI

interface AgentAPI(source)

用于管理单个 agent 实例的 api。

大部分方法都重新返回自身（AgentAPI）以支持链式调用。

关于 agent 上下文的基本概念（如“轮次”）请参阅 SessionContextIndex
