send

abstract fun send(content: MessageContent): Delivery(source)

向 agent 发送消息，无论 agent 状态如何，消息将始终进入队列。 若 agent 空闲（AgentStatus.FREE），消息将被立即消费，触发 agent 的事件循环。 否则，队列中的消息将在下一次 LLM 请求，也就是 AgentStatus.THINKING 阶段前被消费。

无论队列中有多少条消息，消费时都会将它们合并，这之中也可能包含由 AutoTweaker 自动发送的系统消息。

使用 send(content).await() 来挂起等待 agent 消费这条消息，并获取合并后的那条 io.github.autotweaker.api.types.session.SessionMessage.User 的 id 用于前端展示。
