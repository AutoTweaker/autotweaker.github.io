TraceRecorder

interface TraceRecorder(source)

记录数据到 Traces 数据库，通常用于记录有调试价值的原始数据，不要当日志 API 用。

AutoTweaker 会自动清理旧的数据，但并非实时监控，避免每分钟几个 GB 几万条这么存。
