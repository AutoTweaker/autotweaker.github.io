value class CatchingResult<T>

Result 的包装，泛型 T 在类级别，提供成员函数来弥补 Result 错误处理方面的缺陷，并转发 Result 的标准 API。 为了避免 <E, T> 导致编译器无法推断泛型类型，包装 Result，成员函数就不需要 T 了。
